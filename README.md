# Tree-sitter grammar for the GNU MathProg language

[![License](https://img.shields.io/github/license/smoeding/tree-sitter-mathprog.svg)](https://raw.githubusercontent.com/smoeding/tree-sitter-mathprog/master/LICENSE)
[![Build Status](https://github.com/smoeding/tree-sitter-mathprog/actions/workflows/ci.yaml/badge.svg)](https://github.com/smoeding/tree-sitter-mathprog/actions/workflows/ci.yaml)

GNU MathProg is the modeling language used by the [GNU Linear Programming Kit](https://www.gnu.org/software/glpk/) to describe linear programming models. This project uses the [Tree-sitter](https://tree-sitter.github.io/) framework to implement a parser for the language.
